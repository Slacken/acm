机器人的项链

Time Limit: 1000/1000 MS (Java/Others)    Memory Limit: 32768/32768 K (Java/Others)
Total Submission(s): 32    Accepted Submission(s): 16


Problem Description
这个项链有n个的珠子组成，珠子的类型有m种，请问能组成多少种不同类型的项链（若一个项链可以通过另一个项的链旋转得到，那么认为这两个项链为同一种项链）。
 

Input
对于每组数据输入两个正整数n,m(1<n<2^31,0<=m<100)。
 

Output
对于每组数据输出一个整数表示项链的种数。这可能是一个很大的数请对9937取模。
 

Sample Input
10 2
100 99
 

Sample Output
108
843
 

Source
HDOJ 2008 Summer Exercise（2）- Hold by Captain Xu
