题目1168：字符串的查找删除

时间限制：1 秒

内存限制：32 兆

特殊判题：否

提交：225

解决：68

题目描述：

给定一个短字符串（不含空格），再给定若干字符串，在这些字符串中删除所含有的短字符串。

输入：

输入只有1组数据。
输入一个短字符串（不含空格），再输入若干字符串直到文件结束为止。

输出：

删除输入的短字符串(不区分大小写)并去掉空格,输出。

样例输入： 
in
#include 
int main()
{

printf(" Hi ");
}
样例输出： 
#clude
tma()
{

prtf("Hi");
}
提示：

注:将字符串中的In、IN、iN、in删除。
